# U.S. Household Financial Stress Analysis

## Project Overview
This project analyzes how inflation, wages, interest rates, consumer debt, and labor market conditions interact to influence household financial stress over time using U.S. economic data.

## Data Sources
Data sourced from the Federal Reserve Economic Data (FRED) database.
Credit card delinquency data is reported quarterly and was preserved at its native frequency. A monthly-aligned version was created via quarter-level forward filling for visualization purposes only.

## Tools
SQL (MySQL), Tableau Public, GitHub
